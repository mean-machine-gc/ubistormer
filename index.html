About Ubistormer

EventStorming visualization tool for the Ubi framework. This project shows the webhook system architecture using EventStorming methodology.

  • EventStorming Graph: Interactive force-directed graph with filtering and custom layouts
  • Process Flow: Clean Mermaid flowchart generated from the timeline layout

Tip: Arrange nodes in the EventStorming timeline layout, then generate process flow!
